3. Bashkir

5. Belarusian
6. Bulgarian
7. Buryat
8. Chuvash
9. Croatian
10. Kazakh
11. Kyrgyz
12. Kurdish
13. Lezgian
14. Macedonian
15. Montenegrin
16. Romani
17. Russian
18. Rusyn
19. Serbian
20. Tatar
21. Turkmen
22. Ukrainian
23. Uzbek

More will be added in the future.
